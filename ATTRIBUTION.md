## English

This project is licensed under the **MIT License with Attribution Requirements**.

### Requirements for Modified Versions

If you create a modified or derivative version of this software, you **MUST**:

1. **Include a clear reference** to the original source repository in:
   - Documentation (README, help text, website, etc.)
   - Application output (version information, help command, about section)
   - Any distribution materials or derivative works

2. **Original source**: https://github.com/xman12/go-composer

3. **Clearly indicate** that changes have been made to the original software

4. **Do not misrepresent** the origin of the software or claim it as entirely your own work

5. **Include the original copyright notice**:
   ```
   Original work Copyright (c) 2025 Aleksandr Belyshev
   https://github.com/xman12/go-composer
   ```

### Example Attribution

#### In README or Documentation:
```markdown
This project is based on go-composer by Aleksandr Belyshev
Original source: https://github.com/xman12/go-composer
Modifications: [describe your changes]
```

#### In Application Help/Version Output:
```
go-composer-fork v1.1.0
Based on go-composer by Aleksandr Belyshev (https://github.com/xman12/go-composer)
Modified by: [Your Name]
```

#### In Source Code:
```go
// Based on go-composer by Aleksandr Belyshev
// Original: https://github.com/xman12/go-composer
// Modifications: Added feature X, improved Y
```

## Русский

Этот проект распространяется под лицензией **MIT с требованиями атрибуции**.

### Требования к модифицированным версиям

Если вы создаете модифицированную или производную версию этого ПО, вы **ОБЯЗАНЫ**:

1. **Включить четкую ссылку** на оригинальный репозиторий в:
   - Документацию (README, справка, сайт и т.д.)
   - Вывод приложения (информация о версии, команда help, раздел "О программе")
   - Любые дистрибутивы или производные работы

2. **Оригинальный источник**: https://github.com/xman12/go-composer

3. **Явно указать**, что были внесены изменения в оригинальное ПО

4. **Не искажать** информацию о происхождении ПО и не заявлять его как полностью свою работу

5. **Включить оригинальное уведомление об авторских правах**:
   ```
   Оригинальная работа Copyright (c) 2025 Aleksandr Belyshev
   https://github.com/xman12/go-composer
   ```

### Примеры атрибуции

#### В README или документации:
```markdown
Этот проект основан на go-composer, разработанном Aleksandr Belyshev
Оригинальный источник: https://github.com/xman12/go-composer
Модификации: [опишите ваши изменения]
```

#### В выводе приложения (help/version):
```
go-composer-fork v1.1.0
На основе go-composer от Aleksandr Belyshev (https://github.com/xman12/go-composer)
Модифицировано: [Ваше имя]
```

#### В исходном коде:
```go
// Основано на go-composer от Aleksandr Belyshev
// Оригинал: https://github.com/xman12/go-composer
// Модификации: Добавлена функция X, улучшено Y
```
